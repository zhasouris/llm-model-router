# Discussions

Open, **undecided** explorations — distinct from [`../decisions/`](../decisions), which
records *accepted* decisions (ADRs).

A file here captures a live design thread: the context, the options weighed, the tensions,
and the open questions — without committing to an outcome. When a discussion resolves, it
graduates into an ADR (and this note can link to it).
